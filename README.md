# ZOZOWebApp

## 概要
- 「カラーコーディネーター」




## 開発環境の構築手順
1. node.jsのインストール。  
    [ダウンロード | Node.js](https://nodejs.org/ja/download/)  
    10.16.0を使用して開発。
1. コマンド実行
    - Windows PoweShell  
      `cd /ZOZOWebApp;npm i;`  
      `cd ./api;npm i;`
    - Windows cmd  
      `cd /ZOZOWebApp&npm i`  
      `cd ./api&npm i`
    - Linux  
      `cd /ZOZOWebApp && npm i`  
      `cd ./api && npm i`

## Webサーバー実行手順
1. `cd /ZOZOWebApp/api`
1. `node ./bin/www`

## テスト実行
1. `cd /ZOZOWebApp`
1. `npm test`
